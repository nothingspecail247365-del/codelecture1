# Lotto Number Generator

## Overview

A simple web application that generates and displays 5 sets of 6 random lottery numbers.

## Design and Features

### Style and Design
*   **Aesthetics:** A clean and simple design.
*   **Layout:** Centered content, easy to read.
*   **Color Palette:** A playful and exciting color scheme.
*   **Typography:** Clear and large font for the numbers.

### Features
*   **Number Generation:** Generate 5 sets of 6 unique random numbers between 1 and 45.
*   **Display:** Display the generated numbers in a clear and visually appealing way.

## Current Plan

*   **HTML:** Create the basic structure with a title, a button to generate numbers, and a container to display them.
*   **CSS:** Style the application with a simple and clean design.
*   **JavaScript:** Implement the logic to generate and display the random numbers.
